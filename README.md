# AO Hash Function Security Research Wiki

Research wiki on the security of arithmetization-oriented (AO) hash functions,
hash functions designed for efficient evaluation inside arithmetic circuits,
zero-knowledge proof systems, and other algebraic settings.

## Primitives Covered

- Poseidon / Poseidon2
- Rescue / Rescue-Prime
- Griffin
- Anemoi
- Tip5
- Monolith
- MiMC
- GMiMC

## Topics

- Groebner basis attacks
- Algebraic cryptanalysis techniques
- Security parameter selection

## Development

```bash
# Install dependencies
make setup

# Run dev server
make dev

# Build for production
make build

# Format markdown
make format
```
